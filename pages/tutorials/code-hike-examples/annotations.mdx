# Annotations

There are a few more annotations that can be used to highlight code.

### `mark` annotation

```js mark=1[10:14]
function lorem(ipsum, dolor = 1) {
  const sit = ipsum == null ? 0 : ipsum.sit;
  dolor = sit - amet(dolor);
  return dolor;
}
```
