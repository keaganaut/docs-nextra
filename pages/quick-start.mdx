# Quick Start

Welcome to the Quickstart tutorial! The goal of this tutorial is to build and deploy a basic data pipeline in 15 minutes.

Here's an overview of what we're going to do:

1. Create a space
2. Ingest data into BigQuery from a .csv file
3. Transform data using SQL or low-code
4. Schedule a run
5. Visualize the results

## Prerequisites

- Access to an organization in Y42
- Google Cloud BigQuery
    - Billing enabled
    - Service account key in JSON format with the following roles:
        - `BigQuery Admin`
        - `Storage Admin`
        - `Storage Object Admin`
