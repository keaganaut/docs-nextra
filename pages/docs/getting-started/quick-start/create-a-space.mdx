# Step 1 - Create a space
<Callout emoji="ℹ️">
    Skip this section if you wish to do the tutorial in an existing space
</Callout>


To start, create a space in Y42.
1. Select **Add Space** in the space selection page or from top-left drop-down menu in any existing space
2. Enter a space name, e.g. “`Quickstart tutorial`”
3. Select **BigQuery**
4. Upload a BigQuery service account key .json file (note the prerequisites written above)
5. Choose a hosting location/region, where Y42 will create a BigQuery dataset and Google Cloud Storage bucket
6. Select Continue → Complete setup
